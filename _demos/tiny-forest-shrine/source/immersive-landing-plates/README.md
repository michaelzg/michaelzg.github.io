# Immersive Landing Plates

These plates were created with Codex's built-in ImageGen workflow as the art
direction and delivery layer for the Tiny Forest Shrine scroll film. Desktop
plates are native 16:9 compositions (1672 × 941); mobile plates are separately
composed native 9:16 scenes (941 × 1672), not crops.

## Shared prompt direction

Create one chapter of the same original handcrafted floating-island world: a
matte, painterly miniature with watercolor color separation, soft rough
materials, carved stone, moss, warm lantern light, drifting petals and glowing
dust. Preserve deep black negative space for editorial copy and show a physical
route entering and exiting the scene so adjacent chapters feel connected. No
text, logos, copied characters, modern architecture, sci-fi, glossy plastic, or
photorealism.

For mobile variants, recompose the chapter vertically with the path entering
from the previous chapter near the top and exiting toward the next chapter near
the bottom. Preserve safe negative space for copy rather than cropping the
desktop frame.

## Chapter direction

1. `01-threshold`: a lantern-lined stone ascent to a torii gate above golden
   clouds.
2. `02-lantern-crossing`: an uneven suspended path and red arched bridge,
   flowing from the threshold toward water.
3. `03-moon-pond`: a turquoise koi pond, lilies, lantern and tiny waterfalls.
4. `04-ancient-sakura`: a monumental old sakura canopy bridging amber dusk and
   indigo twilight.
5. `05-spirit-grotto`: mossy rocks, luminous mushrooms, cool water and shy
   original pebble-sized forest spirits.
6. `06-awakened-shrine`: the final wooden shrine illuminated by lanterns,
   fireflies and a restrained magical awakening.

The PNG files in this directory are the canonical editable masters. The
`encode_immersive_landing.sh` pipeline derives the responsive WebP posters and
scroll-scrubbed H.264 clips committed under `assets/demos/`.
