# Blog by michaelzg

[michaelzg.com](https://michaelzg.com/). Uses [Jekyll](https://jekyllrb.com/).

## Local development

Prereq: [Docker](https://www.docker.com/).

```bash
make serve
```

Then open http://localhost:4000/.

## Build static site

```bash
make build
```

Generated files are written to `_site/`.
